# Fake Product Identification Using Blockchain & QR Code

## Overview
This project helps detect counterfeit products using QR codes linked to blockchain records.

## Features
- Register product
- Generate QR code
- Scan and verify authenticity
- Blockchain-ready smart contract

## Tech Stack
- Solidity
- Node.js
- Express
- QRCode
- HTML/CSS

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
Open index.html in browser
